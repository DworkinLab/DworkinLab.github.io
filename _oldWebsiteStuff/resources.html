Programs We Like

We like these programs. They help us with our research.

Bowtie - An ultrafast memory-efficient short read aligner.

BWA - Burrows-Wheeler Aligner.

SAMtools - SAM Tools provide various utilities for manipulating alignments in the SAM format.

Gene Palette - GenePalette is a powerful cross-platform and cross-species desktop application for genome sequence visualization and navigation.

ImageJ - General Use Image manipulation program used to aquire landmarks.

Integrative Genomics Viewer - Program to view genomic data in an easy to use GUI.
